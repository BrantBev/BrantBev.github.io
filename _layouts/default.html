Unreal Engine Rendering

This project consists of multiple renderings of a large rendering of a castle scene created in Unreal Engine 5 alongside additional work in Maya, ZBrush, and Photoshop. It included work modeling, rigging, animation, UV editing, and texturing as well as general knowledge in Unreal Engine for composition.

Conway's Game of Life

This project was created in QTCreator and focused on educating the user on John Conway's Game of Life and the idea of cellular automata in general. It allows the user to experiment and play with the rules of Conway's Game of Life. It also allows them to use a few popular known structures as well as save the space to return to or experiment with later.

Board Game Prototype

This project was a single-player board game I developed over several months, stretching from initial brainstorming and ideation, the intial mechanical concepts, the design of multiple printable components, and many cycles of playtesting and revision. The final product is a practically finished board game in its design and mechanics that is both engaging and unique.
